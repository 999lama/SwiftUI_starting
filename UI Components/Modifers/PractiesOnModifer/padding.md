#### .padding() 

```
   var body: some View {
        VStack {
            Text("Hello world")
        }
        .padding()
        .padding(.vertical, 20)
    }
````

* Add a padding to your view if you pass 5 inside the arments will apply the padding to all the corner if you speicfy the padding exmaple; .padding(.vertical, 5) will only give a vertical padding.
